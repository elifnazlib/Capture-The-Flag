# One Flag, No Friends

3D WebGL2 capture the flag game developed with JavaScript, Three.js and Vite.

<img width="512" height="256" alt="gameplay gif" src="https://github.com/elifnazlib/Capture-The-Flag/blob/main/ctf_gameplay.gif" />

## Features of the game

You are controlling a beach ball to capture the flag before your enemy takes it and try to bring it to your base to score. First to score 5 wins the game.

- Move with `W, A, S, D` keys.
- Look around with `mouse`.
- Tag your enemy with `left mouse click`.
- Sprint with `left shift` key.
- Open help menu with `H` key.
- Open settings menu with `P` key.

## How to run:

Node.js must be installed on your system to run this project.

- Download the ZIP file or clone the repository.
- Open the project directory.
- Install the required dependencies through ``npm install`` command.
- Start the development server through ``npm run dev`` command.
- Open the local URL displayed in the terminal: ``http://localhost:5173``
